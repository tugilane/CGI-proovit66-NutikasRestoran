# 2026 CGI suvepraktika proovitöö - Nutikas restorani reserveerimissüsteem

**Käivitamiseks vajaminev tarkvara:** <br>

- Node.js <br>
- Java <br>
- Gradle <br>
  <br>

**Juhised backendi käivitamiseks:** <br>

1. Liigu käsureal kausta "/CGI-proovit66-NutikasRestoran/NutikasRestoran-service <br>
2. Kinnita, et projekt kasutaks Java versiooni ... ning Gradle versiooni ... <br>
3. Jooksuta käsureal käsk "./gradlew bootRun" <br>
   <br>

**Juhised frontendi käivitamiseks:** <br>

1. Liigu käsureal kausta "/CGI-proovit66-NutikasRestoran/NutikasRestoran-web <br>
2. Jooksuta käsureal käsk "npm install" <br>
3. Jooksuta käsureal käsk "npm start" <br>
   <br>

**Frontend lahenduse ja backend lahenduse kohta infot täpsemalt vastavalt web ja service kaustades.**
